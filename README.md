# JS Unit Testing

## Prerequisites
1. NODE version >=6.9.1
2. NPM version >=3.10.8
3. Gulp (installed globally: ` > npm i -g gulp` )
4. ESLint (installed globally: ` > npm i -g eslint` )
5. Git

## Run tests
```
npm test
```
### will create a `report` folder with:
* `coverage` output
* `junit` output

## Run tests coverage
```
npm run test:coverage
```

## Run lint
```
npm run lint
```

## Run mocha tests
```
npm run test-mocha
```
